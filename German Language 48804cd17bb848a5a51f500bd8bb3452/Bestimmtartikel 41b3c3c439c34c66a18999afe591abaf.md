# Bestimmtartikel

definite article

der, die, das → this

# Unbestimmtartikel

indefinite article

(der, das) ein → a

(die) eine → a

 

|  | masculine | feminine | neutral | plural |
| --- | --- | --- | --- | --- |
| bestimmtartikel | der | die | das | die |
| unbestimmtartikel | ein | eine | ein |  |

### Examples

Das ist der stuhl

das ist ein stuhl

das ist das papier

das ist ein papier
